# AI 반응형 웹사이트

현대적이고 반응형인 AI 웹사이트입니다. 바닐라 JavaScript를 사용하여 구현되었습니다.

## 특징

- ✅ 완전 반응형 디자인 (모바일, 태블릿, 데스크톱)
- ✅ 부드러운 스크롤 애니메이션
- ✅ 인터랙티브한 UI 요소
- ✅ 모바일 친화적인 네비게이션
- ✅ 현대적인 그라디언트 디자인
- ✅ 접근성 고려

## 파일 구조

```
AISite/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 기능
├── backend         # AI 채팅용 백엔드 서버
└── README.md       # 프로젝트 문서
```

## 주요 기능

- **네비게이션 바**: 고정된 네비게이션 바와 모바일 햄버거 메뉴
- **히어로 섹션**: 애니메이션이 포함된 메인 배너
- **소개 섹션**: 통계와 함께하는 회사 소개
- **기능 섹션**: 주요 기능을 카드 형식으로 표시
└── *AI 채팅* : AI와 채팅을 하여 정보를 주고 받을 수 있는 공간
- **연락처 폼**: 사용자 입력 폼

## 브라우저 지원

모든 현대적인 웹 브라우저를 지원합니다:
- Chrome (최신)
- Firefox (최신)
- Safari (최신)
- Edge (최신)

## 커스터마이징

`styles.css` 파일의 CSS 변수를 수정하여 색상 테마를 변경할 수 있습니다:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* 다른 색상 변수들... */
}
```
